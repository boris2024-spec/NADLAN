    Backend->>MongoDB: Verify user
    Backend-->>User: Redirect to login
    
    User->>Frontend: Login
    Frontend->>Backend: POST /api/auth/login
    Backend->>MongoDB: Check credentials
    Backend-->>Frontend: Access + Refresh tokens
    Frontend->>Frontend: Store tokens
